# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1hPg1dVcxzrahBZ8dX7WuxQDXfWy_XWjm

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `DOUBAO_API_KEY` in [.env.local](.env.local) to your 豆包 (Doubao) API key
3. Set the `DOUBAO_ENDPOINT_ID` in [.env.local](.env.local) to your 豆包 endpoint ID
3. Run the app:
   `npm run dev`
